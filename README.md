# 🚀 VIP Engineer

> A Full-Stack Notes Sharing & Selling Platform for Engineering Students

🌐 Live Demo: https://vip-engineer.vercel.app

---

## 📖 Overview

VIP Engineer is a full-stack educational platform where students can discover, upload, purchase, and access study notes. The platform includes secure authentication, role-based access, PDF management, payment integration, and an admin dashboard for content management.

---

## ✨ Features

### 👨‍🎓 Student Features

- 🔐 Secure Signup & Login
- 📚 Browse Premium Notes
- 💳 Purchase Notes
- 📄 View Purchased PDFs
- 👤 Update Profile
- 📱 Responsive UI

### 👨‍💼 Admin Features

- ➕ Add New Notes
- 🗑️ Delete Notes
- 📂 Manage Subjects & Folders
- ✅ Verify Purchases
- 📊 Manage Users & Content
